# Font source

**Pixelify Sans** by the Pixelify Sans Project Authors
https://github.com/eifetx/Pixelify-Sans

- Files used: `PixelifySans-Regular.ttf`, `PixelifySans-Bold.ttf`, fetched
  2026-07-12 from the repo's `fonts/ttf/` directory.
- License: SIL Open Font License 1.1 (full text in `OFL.txt` in this
  directory) - free to embed/bundle in this application.
- Not Stardew Valley's actual in-game font (that font isn't openly
  licensed for redistribution) - chosen as a stylistically close, legally
  clean pixel-art font: blocky, rounded pixel letterforms that read as
  "cozy pixel game UI" without claiming to be an exact reproduction.

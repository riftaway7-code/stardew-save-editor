Getting your save file

💻 PC / Mac / Linux (easiest)

Your save is just a file in a folder. Grab it, edit it here, drop it back.

  • Windows: %AppData%\StardewValley\Saves
  • Mac: ~/.config/StardewValley/Saves
  • Linux: ~/.config/StardewValley/Saves

Open your farm's folder — the save is the file named like your farm (no extension). Upload it above, edit, download, copy it back (overwrite). Back up the original first.

📱 iPhone / iPad

iOS keeps the save locked inside the app. A one-click helper to pull it out and push it back is coming soon.

⚠️ Always back up your original save before overwriting. This editor changes only the values you edit and leaves the rest of the file byte-for-byte identical — but a backup means a typo can never cost you a game.
